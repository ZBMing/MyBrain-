# 🚀 快速上手

> 🏠 返回：[[home|主页]]

## 第一步：打开你的 Vault

1. 启动 Obsidian
2. 点击「打开本地仓库」(Open folder as vault)
3. 选择文件夹：`~/Obsidian/MyBrain`
4. 看到 🏠 Home 页面，就对了 ✅

## 第二步：试试这些操作

| 操作 | 怎么做 | 效果 |
|------|--------|------|
| 新建笔记 | `Cmd+N` | 自动存到 Inbox，稍后整理 |
| 快速切换 | `Cmd+O` | 输入文件名，瞬间跳转 |
| 链接笔记 | 输入双方括号 `[[ ]]` | 创建双向链接 |
| 今日日记 | 点左侧日历图标 📅 | 从模板自动创建 |
| 使用模板 | `Cmd+T` 或左侧插入模板 | 选模板快速启动 |
| 命令面板 | `Cmd+P` | 所有功能入口 |
| 知识图谱 | `Cmd+G` | 可视化你的笔记网络 |

## 第三步：今天就记第一条

打开 [[📥 收件箱|Inbox]]，记下你此刻在学/在想的东西：

- 正在学的技术 / 框架？
- 最近读的书 / 文章？
- 工作中遇到的一个问题？

**格式随意，先记下来就是胜利。**

## 推荐首次安装的插件

打开 设置 → 第三方插件 → 关闭安全模式 → 浏览社区插件：

1. **Dataview** — 把笔记变成可查询的数据库
2. **Calendar** — 日记日历（配合每日笔记）
3. **Tag Wrangler** — 标签管理
4. **Excalidraw** — 手绘图表
5. **Paste URL into selection** — 粘贴链接自动转标题

> 💡 先别装太多，用一周再加。插件是调味料，不是主菜。

## 每日最小行动

```
早晨/开始工作前：Cmd+N → 记今日重点
全天：想到什么，Cmd+N 扔 Inbox
晚上/收工前：花 2 分钟回顾今天记了什么
周日：20 分钟清理 Inbox，归入 PARA
```

> **记住：完美是执行的敌人。一条乱七八糟的笔记，胜过一条从未写下的完美笔记。**
