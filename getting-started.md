# 🚀 快速上手

> 🏠 返回：[[home|主页]]

## 第一步：打开你的 Vault

1. 启动 Obsidian
2. 点击「打开本地仓库」(Open folder as vault)
3. 选择文件夹：`~/Obsidian/MyBrain`
4. 看到 home 仪表盘页面，就对了 ✅

---

## 第二步：安装增强插件

打开 设置 → 第三方插件 → 关闭安全模式 → 浏览社区插件：

### 已安装 ✅

| 插件 | 作用 |
|------|------|
| **Dataview** | 把笔记变成可查询的数据库 |
| **Calendar** | 日记日历（配合每日笔记） |
| **Tag Wrangler** | 标签管理 |
| **Excalidraw** | 手绘图表 |
| **Paste URL into selection** | 粘贴链接自动转标题 |

### 建议安装 📦

| 插件 | 搜索关键词 | 作用 | 优先级 |
|------|----------|------|:--:|
| **Obsidian Git** | `obsidian-git` | 每次保存自动提交+推送到GitHub | ⭐⭐⭐⭐⭐ |
| **Tasks** | `tasks` | 跨所有笔记聚合任务到统一面板 | ⭐⭐⭐⭐⭐ |
| **Kanban** | `kanban` | 把学习路线变成拖拽看板视图 | ⭐⭐⭐⭐ |
| **Spaced Repetition** | `spaced-repetition` | 核心概念/术语自动提醒复习 | ⭐⭐⭐⭐ |
| **Quick Add** | `quickadd` | 一键选择模板创建笔记 | ⭐⭐⭐ |
| **Homepage** | `homepage` | 每次启动 Obsidian 自动显示仪表盘 | ⭐⭐⭐ |

> 💡 已为 7 篇核心概念笔记添加了 Spaced Repetition 标记（间隔3天复习），安装插件后即可使用。

---

## 第三步：核心操作

| 操作 | 方式 |
|------|------|
| 新建笔记 | `Cmd+N` → 自动存入 Inbox |
| 快速跳转 | `Cmd+O` → 输入笔记名 |
| 创建双向链接 | 输入 `[[ ]]` |
| 今日日记 | 点左侧日历图标 📅 |
| 使用模板 | `Cmd+T` |
| 命令面板 | `Cmd+P` |
| 知识图谱 | `Cmd+G` |
| 快速捕获（终端） | `cap "灵感想法"` |

---

## 第四步：今天就记第一条

打开 [[📥 收件箱|Inbox]]，记下你此刻在学/在想的东西。

**格式随意，先记下来就是胜利。**

---

## 每日最小行动

```
🌅 早晨：Cmd+N → 今日笔记 → 写今日重点
☀️ 全天：Cmd+N → 新概念 → 记到 Inbox
☕ 随时：终端 cap "突然的想法"
🌙 晚上：终端 brain-sync（一键备份+同步）
📅 周日：20 分钟清理 Inbox，归入 PARA
```

> **记住：完美是执行的敌人。一条乱七八糟的笔记，胜过一条从未写下的完美笔记。**
