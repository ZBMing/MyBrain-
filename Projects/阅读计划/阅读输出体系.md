# ✍️ 阅读输出体系

> 🎯 读了128本书却没有输出=吃了128顿饭没有消化
> 📖 配套：[[五年阅读人生计划|五年阅读计划]]

---

## 🧠 核心理念

```
输入:输出 = 3:1

每读3小时 → 输出1小时
每读1本书 → 至少产出1篇笔记
每读10本书 → 至少产出1篇长文

不输出的阅读=浏览。输出的阅读=内化。
```

---

## L1 · 3句话笔记（每本必做）

> **工具：** Obsidian 日记 | **时机：** 读完当天
> **存储：** `Daily/` 日记中，或独立笔记 `Resources/阅读笔记/`

### 模板

```markdown
## 📖 《书名》· 读完 | 2026-XX-XX

1. **讲了什么：** （20字以内概括）
2. **最触动我的一个点：** （为什么打动你？联想到自己的什么经历？）
3. **如果只能推荐给一个人，我会推荐给____，因为____**
```

### 示例

```markdown
## 📖 《局外人》· 读完 | 2026-07-15

1. **讲了什么：** 一个对一切漠不关心的职员，因为太阳太晒而杀人，被审判时却因为"在母亲葬礼上没有哭"而被判死刑
2. **最触动我的：** 默尔索在死牢里对神父爆发的那段——他终于确认了自己活过的方式是对的。他不妥协，哪怕以生命为代价。这让我想到自己有时候"融入群体"的疲惫
3. **推荐给：** 所有觉得"社交好累"的人——这本书会告诉你，不合群不是病
```

## L2 · 300字微书评（从第3个月开始）

> **频率：** 每2本书写1篇 | **发布：** 先在 Obsidian 里写，第6个月开始发知乎
> **存储：** `Resources/阅读笔记/微书评/`

### 300字结构

```
开头（50字）：一个钩子——为什么你要读这篇书评
中间（200字）：一个核心观点+一个具体例子
结尾（50字）：这本书改变了我对____的看法
```

### 为什么从第3个月开始

前12本书只写L1。先把"读完就写3句话"练成肌肉记忆，再加码。

## L3 · 概念笔记（第2年开始）

> **频率：** 每本值得的书至少提炼1个概念 | **存储：** `Resources/知识点/`
> **与现有知识库联动：** 每个概念链接到已有的技术/心理学概念

### 模板

```markdown
---
tags: [阅读, 概念, 书名标签]
created: YYYY-MM-DD
source: "《书名》"
---

# 概念名（你自己起的名字）

> 一句话定义

## 书里怎么说的
（引用+你的理解）

## 和技术/工作的关联
（这个概念怎么帮你理解CV/AI/工程问题？）

## 和生活的关系
（这个概念在你生活中出现过吗？）

## 相关
- [[已有概念1]]
- [[已有概念2]]
```

### 示例：从《反脆弱》提炼"杠铃策略"

```markdown
# 杠铃策略

> 把90%的资源放在极度安全的地方，10%放在极度冒险的地方——避开中间

## 和技术的关系
- 技能组合：OpenCV（安全区）+ VLA前沿（冒险区）
- 学习策略：工作日学确定有用的 + 周末探索不确定但有趣的

## 和生活的关系
- 职业选择：主业保底 + 开源/Kaggle/写作赌上限
```

## L4 · 主题长文（第3年开始）

> **频率：** 每季度1篇 | **3000-5000字**
> **发布：** 知乎专栏 + 个人博客

### 选题来源

```
读完10本关于"人为什么痛苦"的书
    ↓
写一篇："我读了10本心理学书，发现痛苦只有三个来源"
    ↓
这不是读书笔记，这是你的原创观点
书是你的素材，观点是你自己的
```

### 选题公式

| 公式 | 示例 |
|------|------|
| "我读了N本关于X的书，发现____" | 读了8本传记，发现成功者只有两个共同点 |
| "从A到B：一个XX的阅读之旅" | 从《活着》到《存在主义》：我用一年时间理解了死亡 |
| "为什么XX经典在今天更重要" | 为什么1984在AI时代比1949年更可怕 |
| "如果你只想读一本关于XX的书" | 如果你只想读一本心理学书，读这本 |

## 🖊️ 仿写训练（第2-5年，贯穿全程）

> **理念：** 最好的写作老师是你最喜欢的作家
> **频率：** 每月1次，选一段你读完觉得"写得真好"的文字

### 方法

```
1. 选一段200-300字的文字（你读完觉得"我永远写不出这种句子"）
2. 手抄一遍——感受每个字的位置
3. 用自己的话改写——保持同样的结构、节奏、情感
4. 对比原文——差距在哪里？是词汇？是节奏？是观察角度？
5. 尝试用这个风格写一个全新的主题
```

### 五年仿写进阶

| 年 | 仿写对象 | 学习重点 |
|:--:|------|------|
| 1 | — | 先读，不仿写 |
| 2 | 毛姆、王小波 | 简洁有力，讽刺幽默 |
| 3 | 马尔克斯、沈从文 | 画面感，魔幻与现实 |
| 4 | 鲁迅、奥威尔 | 思想的锋利度 |
| 5 | 找到你自己的声音 | 不再模仿，自由表达 |

## 🎯 为什么输出如此重要

```
读《活着》哭了 → 过一周忘了                       ← 没有输出
读《活着》哭了 → 写了3句话 → 过一周还记得福贵       ← L1
读《活着》哭了 → 写了300字 → 理解了"活着本身就是意义" ← L2
读《活着》+《局外人》+《西西弗神话》→ 写出了自己对"荒诞"的理解 ← L4

输出层次越高，书在你身上留下的痕迹越深。
读100本书不输出 → 你是图书馆
读50本书深度输出 → 你是作者
```
