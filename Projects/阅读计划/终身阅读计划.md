## 🧠 为什么需要终身阅读

```
你正在学机器视觉——一个技术迭代以月为单位的领域。
如果不读书，5年后你的技术可能过时。
如果只读技术书，你的思维会越来越窄。

阅读文学作品 → 理解人性（AI 最终是要服务人的）
阅读历史哲学 → 看清大势（才知道该往哪个方向努力）
阅读科学元典 → 掌握第一性原理（不被二手知识忽悠）

技术决定你能走多快，阅读决定你能走多远。
```

# 🔖 纪律与技巧

## 铁律

1. **每天摸一下书**：哪怕只读1页，保持触觉记忆
2. **同时2-3本并行**：主菜+轻松副线+技术书（不冲突）
3. **5分钟撤退法**：翻开读5分钟还不想要就换
4. **不补过去的**：断档不焦虑，今天开始
5. **每本3句话**：讲了什么/最触动/推荐给谁

## 与CV学习的时间分配

```
工作日：CV学习 3-4h | 阅读 15-30min（睡前）
周末：  CV学习 6-8h | 阅读 30-45min（早晨或午后）
```

阅读是**充电**，不是**负担**。CV学习累了读小说=换脑休息。
