# 📚 12周阅读习惯养成计划

> 🎯 目标：从"不怎么读书"到"每天不读点什么总觉得少了什么"
> 📖 来源：[[阅读路线图_渐进式]] · 🟢 入门级 40 本
> ⚡ 原则：**先爱上阅读，再让阅读改变你**

---

## 🧠 核心理念

```
习惯 > 数量 > 速度
  ↓
前4周：每天读10分钟就算胜利
中4周：每天读20分钟，开始有"想读"的感觉
后4周：每天读30分钟，不读不舒服
```

## 📅 每周节奏

```
周一：翻开新书，读10页（认识一下）
周二~周五：每天读15-20分钟（不追求页数）
周六：读30分钟+写3句话笔记
周日：本周回顾+决定下周继续还是换书
```

### 每天只需

| 阶段 | 每天 | 大约页数 |
|------|------|:--:|
| W1-W4 | 10分钟 | 5-10页 |
| W5-W8 | 20分钟 | 10-20页 |
| W9-W12 | 30分钟 | 20-30页 |

## 🔖 阅读纪律

| 规则 | 说明 |
|------|------|
| **5分钟撤退法** | 翻开读5分钟，真不想读就换一本——但至少读完正在看的这章 |
| **不补过去的** | 断档了不焦虑，今天直接开始 |
| **允许跳级** | 遇到特别想读的，不管哪个层级，直接读 |
| **手机放远** | 阅读时手机放另一个房间 |
