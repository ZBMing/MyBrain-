# 📚 五年阅读人生计划

> 🎯 **核心命题：** 如果一个机器视觉工程师，坚持深度阅读五年，他会变成什么样的人？
> ⏱ 2026-2031 | 📖 ~128本精选 | 🎓 5个年度主题
> ⚡ **今天开始。不是明天，不是下周。是现在。**

---

## 🧬 设计哲学

```
不是"读完多少本书"，而是"成为什么样的人"。

五年后，你早上醒来时——
你看到的不是像素和代码，而是模式和本质；
你遇到的不是"bug"，而是"系统边界的信号"；
你做出的不是技术决策，而是对人和世界的理解；
你不再是"写代码的人"，而是"能定义问题的人"。

阅读不是逃避工作。阅读是你工作的一部分。
```

---

## 🗺️ 五年全景

```
2026-2027 🟢 觉醒之年    "原来书里有另一个世界"     25本   CV实习生→初级工程师
2027-2028 🟡 纵深之年    "我开始有自己的观点了"     28本   初级→中级工程师  
2028-2029 🟠 拓展之年    "一切知识都是相连的"       30本   中级→高级工程师
2029-2030 🔴 融通之年    "我能同时容纳矛盾的观点"   25本   高级→技术专家
2030-2031 ⚫ 表达之年    "我有了想对世界说的话"     20本   开始输出和影响
```

# 🔖 阅读纪律

1. **每天必摸书**：哪怕只读1页，保持触觉记忆
2. **5分钟撤退**：翻开5分钟还不想读，换一本——但读完正在看的这章
3. **3句话笔记**：讲了什么/最触动/推荐给谁
4. **允许跳级**：遇到特别想读的，不管在哪一年，直接读
5. **允许放弃**：一本书读到1/3还不喜欢，果断放下。生命太短，好书太多
6. **不补过去**：断档了不焦虑，今天翻开新的一页就是胜利
