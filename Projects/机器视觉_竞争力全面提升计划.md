## 二、Kaggle竞赛路线图（嵌入每周计划）

### 为什么Kaggle是你最该做的事？

| 维度 | 价值 |
|------|------|
| **简历筛选** | 几百份简历里，"Kaggle Top 10%"能让HR多停30秒 |
| **转行背书** | 非科班出身→Kaggle奖牌=第三方对你CV能力的独立认证 |
| **面试话题** | "我在Kaggle钢铁缺陷检测赛里用了Focal Loss解决类别不平衡" — 秒杀背诵八股文的候选人 |
| **代码公开** | Kaggle Notebook是你的开放式代码简历 |
| **工业关联** | 缺陷检测、图像分割、目标检测赛 → 直接对口你的求职方向 |

### 竞赛分级与目标

```
🥉 铜牌（Top 40%）— 最低目标，认真做就能达到
🥈 银牌（Top 20%）— 需要模型集成+数据增强+后处理优化
🥇 金牌（Top 10%）— 需要创新方法，6个月后再挑战
```
