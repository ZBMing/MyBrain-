## ⚡ 命令速查

### 学习与笔记

| 命令 | 作用 | 示例 |
|------|------|------|
| `ci "W1 Day1" 3 "备注"` | 每日学习打卡 | `ci "W1 Day2" 3 "函数+lambda"` |
| `cv` | 学习仪表盘 | `cv` |
| `cap "内容"` | 快速捕获到 Inbox | `cap "VLA中动态路由是核心创新"` |
| `cap --link "URL" "标题"` | 捕获链接 | `cap --link "https://arxiv.org/..." "新论文"` |
| `brain-sync` | Git提交 + gbrain同步 | `brain-sync` |
| `brain-health` | 系统健康检查 | `brain-health` |
| `brain-stats` | 知识库统计 | `brain-stats` |

### 搜索

| 命令 | 作用 |
|------|------|
| `gbrain query "问题"` | 语义搜索 |
| `gbrain search "关键词"` | 关键词搜索 |

### 趣味工具箱 `bp`

| 命令 | 作用 |
|------|------|
| `bp today` | ☀️ 今日全局面板 |
| `bp question` | 💡 跨领域思考题 |
| `bp wander` | 🎲 随机发现知识点 |
| `bp blindspot` | 🔍 知识盲区检测 |
| `bp inspire` | ✍️ 写作灵感生成 |
| `bp vs "A" "B"` | ⚔️ 概念对决 |
| `bp path "A" "B"` | 🕸️ 六度分隔 |

## 🎮 趣味工具箱

终端输入 `bp` 查看完整帮助。所有工具集成在 `~/bin/brain-play`（499行）。

**最常用的三个：**

```bash
bp today      # 每日必看——打卡+知识库+随机推荐
bp question   # 每天一个跨领域思考题
bp blindspot  # 每周检查知识盲区
```

**生成可视化报告：**

```bash
bp heatmap    # GitHub风格学习热力图 → 📊 学习热力图_2026.html
bp annual     # Spotify Wrapped年度报告 → 📊 年度报告_2026.html
bp wordcloud  # 本周词云 → 📝 本周词云.html
```

**知识探索：**

```bash
bp wander          # 随机发现一个知识点
bp path A B        # 两个概念的最短连接路径
bp vs "GelSight" "CNN"  # 概念对决
bp inspire         # 生成写作灵感+文章框架
```
