人之所以无法下定决心做自己想做的事情，是因为不够自由
1、什么是自由以及为何不自由
自由=责任+意志
意志=愿望+决定

```
责任：成为自我生命中毫无争议的个人原创作者

意志：做出选择并让它执行的能力

  包含让你充满动力的愿望；让你坚定朝向目标前行的决定

  意志是让一个人自律、有决策力、有行动力、有强动力的基本条件之一
```
  很难拥有真正的意志力的四种人：
    对幻想中危险的恐惧者：
    当一个人有愿望时灾难化的联想会让他无法决定和行动
    不愿意澄清内心恐惧的准确性，于是永远止步于这个恐惧
    无常世界的被动接受者：
    被动接受的状态很容易让人陷入到我无法为自己的行为做主的思维模式里，充满埋怨、怨恨和无力
    无力感是无法让一个人决定改变的，它只会让一个人不停的向世界释放攻击性
    对愿望和冲动不加分辨的冲动者：
    生活中很多人认为要遵从本心，活在当下，但却成为了很多人随时随地的冲动行为的借口；
    当一个人决定想做什么就做什么，就会压制对真实愿望的行动力
    被多重期望包裹起来的强迫者：
    依照系统里的某种框架而行动，一旦不这么行动就会感到不安和内疚，外人看来是一个优秀的人是个成功者，但内心却没有升起愿望和热情
    有愿望，但不是自己的愿望，有行动的决定但不是自己的行动的决定
    不敢满足自己的愿望，内心觉得所有的愿望都是被禁止的
```

存在主义心理学家——罗洛·梅：
如果只有愿望而没有行动的决定，人就是被动、不自由、幼稚的就像长不大的婴儿或机器人；如果只有行动而没有愿望，人就是干瘪的，像维多利亚时代的新清教徒

```

2、如何获得自己的意志力
	外部强加的愿望+强加的行动=逆反的愿望+逆反的行为
	真正让人通向意志的方法，不是从外部鞭策出一个新的意志，而是从心里结构中为人们已经存在的意志去除掉障碍

去除心理障碍：
1.被封锁的自我
	  想尽一切办法充分表达和感受自己的愿望，如果愿望不是出自于内心的感受，那就是规训
	  把我不能变为我可以但不愿意；外界怎么怎么样变为我让自己在环境中怎么样
	  或者说话后表达为此负责
	  一个人之所为成为自己而不是别人，就是由一生中无数个自我选择塑造的；如果害怕让别人失望而麻木自己不去选择，不去感受自己的愿望，那也就失去了构建自己的能力

2.互相冲突的需要
	  朝目标前进和放弃是一回事，坚定一个目标必然会放弃另外的相反的目标
	  做出一个决定，就意味着放弃了其他所有可能性作为代价
	  选择的关键在于意识到什么是你作为一个成年人真正想要的东西
	  如果不去做出愿望的选择，这也是一种选择，这是我们为了避免放弃带来的痛苦做出止步不前反复内耗的选择

  思想家、文学家、诗人——拉尔夫·沃尔多·爱默生：
  只有大胆的切断退路、烧断桥梁，我们才能朝着自己的规划进发

3.恐惧障碍
	  打破恐惧的关键是意识到每一个决定都是可以调整的
	  我们需要根据现实情况不断的去修整和调整自己的决定

  暴露行为疗法：
	将目标患者暴露在焦虑或恐惧来源或情形中，在不造成真实伤害的前提下，帮助他们克服焦虑或痛苦情绪，多用于治疗焦虑症

4.元决定障碍
	  需要意识到自己才是生活中大小事务的决定者
	  这世界上并不终极价值、参照物、意义、标准，你就是你的标准
	  这就是责任——你是你生活的原创者
  
  希腊神话中特洛伊统帅阿伽门农之子俄瑞斯特斯：
	昨天我还无目的的行走，踏过成千上万的道路，却哪里也没有走到，因为那些都是其他人的路。今天我只有一条路，谁也不知道这条路通向哪里，但这是我的路。


自由是在挖掘自我热爱的前提下充分的履行对人生的责任，这个责任是主动成为自己世界的个人原创者
