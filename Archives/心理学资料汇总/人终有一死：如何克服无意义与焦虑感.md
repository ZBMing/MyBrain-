1、欧文亚隆：
人一生中的痛苦都有四个小小的来源：
```
1、我们最终会死亡
2、也许生活并无一个显而易见的意义可言
3、我们需要自由，却害怕自由
4、我们内心深处的孤独感
```

2、直视死亡后的生活改变
	1、重新安排生活的优先级，不再重视没有意义的事
	2、释放的感觉、能选择不做自己不想做的事
	3、活在当下的感觉增强，不再拖延到退休或某个未来时间之后过想要的生活
	4、欣赏生命中鲜明生动的自然现象，如四季更替
	5、与所爱之人有更深入的沟通
	6、对人际间的恐惧更少，更不担心被拒绝，更敢于冒险
总计而言就是获得更加真实、无谓且充分

3、死亡是焦虑的原始来源
	死亡对于个体存在自尊的损害无法掌控和消除，所以我们会把它置换到一个可以控制的日常事物上增强掌控感
	可怕处在于这些问题本可以不存在，但却因为回避一个基础事实而存在——人不是无坚不摧的，认识会毁灭的

4、肉体的死亡会毁掉人，但死亡的观念却能让人觉醒
	直视死亡可以将人从一种随波逐流的生存模式换挡到一种追求意义的生存模式上
```
1、消除一些微不足道的恐惧
2、发现微不足道的喜悦
3、审视微不足道的认同
```
  ```乔布斯一生中最重要的箴言：记住我即将死去
  几乎所有的事情，包括外部的期待、荣耀、尴尬和失败都会随着死亡消失
  ```

重点不是在对痛苦的规避，而是不再盲目的活在外部的期待里，用平常心对待外部期待
```
乔布斯：你已经赤身裸体了，你没有理由不去追随本心
```
死亡的观念可以让你放弃一些抓不住的假目标，去寻找自己真正认可的有意义的存在方式

5、如何直面死亡焦虑
	马丁海德格尔：向死而生
	人不是患病才走向死亡，而是从出生的那一刻就开始走向死亡
	向死而生是一种以始为终的极度清醒的存在方式
如何领悟向死而生：
```
1、增强日常生活中对死亡的觉察

2、通过想象与对话深入思考死亡

3、通过边缘体验——无法改变，无法挽回的紧急体验（海德格尔）
```

欧文亚隆-九种引发觉醒的事件：
```
1、亲密关系的破裂

2、失业或更换职业

3、子女离家后突如其来的自由

4、重大创伤如离婚、火灾、破产、抢劫等

5、身边亲爱的人过世

6、一些重要的生命历程碑

7、退休及老人特征的出现

8、患有危及生命的疾病

9、搬至敬老院
```

祁克果：冒险是危险的为什么？因为可能会有所损失。不冒险是聪明的，但不冒险非常荣日丧失即使在最危险的冒险中也很难丧失的东西，就是你自己

6、人生并无既定的意义，我们为什么活着
人的许多无意义的痛苦都来源于没看清楚一件事：
```
意义不太可能直接被找到，但却可以在每个人参与世界的过程中被发现、诠释和创造
```

生命中的重大事件，在当时来看也许是造成你无意义的危机，当你直面了它做出自己的选择、决定和行动，他就会变成你所有意义感的来源

生命中好多痛苦的根源在于，我们都是没有被知会就被抛入这个充满不确定的世界里的。我们要怎么想、怎么活根本没有绝对的参考系。而人一开始对意义的想象，如努力奋斗、享受当下、创造价值、生儿育女等，是别人在他的世界、他的时代里尝试后为你展现出的一种可能性。它可能适合某个阶段的你，但不会永远适合你。于是你会感到不安、烦躁、不满和无助

人与其他存在的不同在于人可以反省自身的存在，这会带来一个赤裸裸，毛骨悚然却又欣喜万分的洞察-你是你意义的创造者

《存在主义咖啡馆》沙拉·贝克韦尔

```
我根本没有预先被决定的本性。我的本性要通过我选择去做什么来创造。我可能会被我的生物性影响，或者被我所处的文化和个人背景等方面影响。但这些并不能合成一张用来制造我的完整蓝图。我总是先我自己一步，边前行边构筑自身。你可能认为你可以用一些标签定义我但是你错了，因为我始终会是一件正在加工的作品。我不断的通过行动创造自身，我是我自己的自由，不多也不少。
```

维克多·弗兰克尔拆盲盒：

```
1、体验——体验某种事或某个人

2、创造——创造某种事情或采取某种行动
```
