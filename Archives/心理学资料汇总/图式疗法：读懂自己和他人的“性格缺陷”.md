图式心理治疗：人不断出现的心理问题汇总成18种模式：
分离和拒绝：
```
1、遗弃/不稳定

2、不信任/虐待

3、情感剥夺

4、社会孤立/疏离
```

自主性和能力不足：
```
6、依赖/无能力

7、对伤害和疾病的易感性

8、纠结/未发展的自我

9、失败
```

限制不足：
```
10、权利/夸张

11、缺乏自控/缺乏自律
```

他人导向：
```
12、屈从

13、自我牺牲

14、寻求称赞/认可
```

过于警惕和压抑：
```
15、消极/悲观

16、情感压抑

17、苛刻标准/吹毛求疵

18、惩罚
```
35岁之前一定要觉察自己的性格缺陷并完成该拜年，否则会一直创造喂养性格缺陷的环境

五种常见图式：
	1、是否时常觉得自己不被自己在乎的人了解、保护、指引或者温柔的对待；是否常常觉得被忽视，很少有人关心你身上发生了什么事——情感剥夺图式
	认为自己的情感需求被剥夺了
	进入情感剥夺的图式会对别人不关心、不在乎自己的蛛丝马迹非常敏感，仿佛不被爱的小透明。无论对方有意或者无意，都会认为自己没有得到情感支持
	根本原因：童年时期真的经历过带来创伤的情感剥夺：
	1、当你处于负面状态时，你的抚育者是否对你缺乏应有的关注、慈爱、温暖或陪伴——养育剥夺
	2、当你有自己的想法或情绪表达时，你的抚育者是否对你缺乏基本的理解、倾听、自我表露和情感的分享——共情剥夺
	3、有问题无法肚子解决时，抚育者是否对你缺乏基本的力量给予、指导和引领——保护剥夺
	基本观念：永远没有人了解自己、所有人都会让自己失望、没有人重视自己
	行为：不表达自己的感受和需求
	后果：找一个和父母相似的伴侣维持图式；在接触中找到不爱的痕迹导致关系破裂
	2、你是否认为自己是个很乏味、很无聊的人，不知道怎么和他人打交道；你是否觉得自己不属于任何地方，感觉和他人格格不入——社交孤立/疏离图式
	3、你是否觉得自己一个人很难处理新的挑战、胜任新的职位，需要别人来帮你度过难关。在生活中，是不是有一个人，比如父母或者你的朋友伴侣，过度干涉你的生活
	4、你是否在职场学校运动等等方面，认为自己没有比多数人有能力；你是否常常觉得自己不够好、没有天赋，在才华智慧或者成就上都比别人差一等
	5、你是否感觉别无选择，只能迎合别人的期待，否则他们会以某种方式拒绝你或者否认你；你是不是会压抑自己的喜好、决定和愿望，常常变现的过渡顺从
