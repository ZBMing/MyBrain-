# 🔀 跨领域交叉

> 心理学 × 机器视觉：两个看似无关领域的深层连接

## 为什么做交叉

- 视觉感知是人类最复杂的认知功能之一 → 直接启发计算机视觉架构
- 认知偏误不仅影响人类决策 → 也会被编码进 AI 系统的训练数据
- 学习科学 → 优化自己的学习方法论

## 交叉主题

### 1. 视觉感知 → 计算机视觉

| 人类视觉 | 机器视觉对应 |
|---------|------------|
| 特征检测（边缘、纹理） | CNN 卷积核 |
| 注意力机制 | Transformer Attention |
| 层级处理（V1→V4→IT） | VLM 多层 backbone |
| 多模态整合（视觉+语言） | VLA 端到端架构 |

核心笔记：[[visual-perception]]

### 2. 认知偏误 → AI 系统设计

- **确认偏误** → 训练数据偏差放大
- **锚定效应** → 模型初始化权重的影响
- **过度自信** → 模型校准（calibration）

### 3. 学习科学 → ML 方法论

- **间隔重复** → 课程学习（curriculum learning）
- **交错练习** → 多任务学习（multi-task learning）
- **精细编码** → 对比学习（contrastive learning）
