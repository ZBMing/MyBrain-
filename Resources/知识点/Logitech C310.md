**Logitech C310** 是一款经典的入门级高清网络摄像头，在 GelSight 触觉传感器的发展历程中，它扮演了关键角色。

在我们之前的讨论中，提到了 GelSight 传感器从实验室走向机器人指尖的过程中，需要解决“紧凑化”和“低成本”的问题，而 **Logitech C310 正是实现这一目标的核心组件之一**。

### 🔬 它在 GelSight 中扮演的角色

你可以在 MIT 关于指尖 GelSight 的论文中，明确看到它的身影。在传感器硬件分解图中，就包含了这台拆去外壳的 Logitech C310 摄像头。

它的作用是**作为传感器的“眼睛”**，负责以高帧率拍摄凝胶表面的形变图像，形成触觉数据源。在 GelSight 团队的早期研究成果里，从改良版 GelSight 传感器到第一代 Fingertip GelSight，**Logitech C310 都是标准的成像方案**。

### 📊 为什么是它？

Logitech C310 能成为 GelSight 研究中的“标准配置”，得益于它在性能、尺寸和成本之间的平衡：

*   **性能足够**：它支持 **720p/30fps** 的视频流，能提供清晰的触觉图像，并且通过 **RightLight™ 2** 技术进行自动光线校正，保证在不同光照条件下图像的质量。
*   **体积小巧**：其紧凑的尺寸让研究者可以将其集成到传感器模组中，不必像早期原型那样使用笨重的单反相机，为 GelSight 的“指尖化”铺平了道路。
*   **成本低廉**：作为一款面向普通消费者的产品，它的价格远低于科研用的工业相机，符合 GelSight 技术走向普惠和普及的需求。

以下是它的关键规格参数：

| 规格项 | 具体参数 |
| :--- | :--- |
| **最大分辨率** | 1280 x 720 (720p HD) |
| **最高帧率** | 30 fps |
| **传感器** | CMOS |
| **接口** | USB 2.0 |
| **视野 (对角)** | 60° |
| **特色功能** | RightLight 2 自动光线校正、内置降噪麦克风 |

需要注意的是，这款型号早在2013年就已经停产。但它在 GelSight 技术从实验室原型走向实用化、指尖化的过程中，是一个重要的里程碑。
