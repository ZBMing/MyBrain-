“非局部自相似性”正是 BM3D 算法以及许多先进图像处理技术的核心思想。理解了它，你就明白了为什么 BM3D 的去噪效果能远超中值滤波这类传统方法。

---

### 🧠 第一层：最直白的定义

**非局部自相似性（Non-Local Self-Similarity）** 是指：在一幅自然图像中，**不同位置、互不相邻的图像块之间，往往存在着相似的纹理、结构或图案**。

**一句话通俗理解**：图像里总有很多“重复”的局部片段——比如砖墙的纹理、草地、织物，或者 GelSight 图像中相似的形变模式。

---

### 🏙️ 第二层：与“局部性”的对比

| 特性 | **局部方法**（如中值滤波） | **非局部方法**（如 BM3D） |
| :--- | :--- | :--- |
| **信息利用** | 只看像素附近的邻域 | 利用整张图像中**所有相似的结构** |
| **去噪逻辑** | 用邻居的平均/中值替换噪声 | **“组团”去噪**：所有相似块一起参与去噪 |
| **对重复纹理的利用** | 无 | **强**（能有效利用重复模式） |
| **计算量** | 低 | **高** |
| **去噪质量** | 中等 | **极高**（目前传统方法的顶级水平） |

**一个核心洞察**：自然图像是高度冗余的——这种冗余不仅来自空间连续性（相邻像素相似），还来自**非局部重复性**（远处也可能有相似的模式）。正是这种冗余，使得 BM3D 能够实现惊人的去噪效果。

---

### 🏗️ 第三层：它的物理直觉与数学本质

为什么非局部自相似性在 GelSight 和真实图像中普遍成立？

| 层面 | 物理直觉 | 数学解释 |
| :--- | :--- | :--- |
| **1. 纹理重复** | 类似的地砖、布料、草地会反复出现。 | 图像中的高频纹理模式是**全局重复的**。 |
| **2. 形变模式** | GelSight 中的“碗状凹陷”在不同位置出现时形状相似。 | 连续的物理场具有**局部相似性**，同一形变模式可能在不同位置复现。 |
| **3. 结构冗余** | 边缘、拐角等结构在图像中大量重复。 | 图像是由基元（Primitives）组成的，同类基元会重复出现。 |
| **4. 降维与低秩性** | 相似块构成的矩阵具有低秩特性。 | 非局部相似性对应了矩阵的低秩结构，因此能用于去噪。 |

---

### 🔧 第四层：非局部自相似性在 BM3D 中的具体应用

BM3D 正是建立在这一洞察之上的：

1.  **块匹配**：对每个“参考块”，在海量候选块中，找到与其**最相似的若干个块**（如通过 L2 距离或相关系数衡量）。
2.  **三维堆叠**：将这些相似的二维块堆叠成一个**三维数组**。每个块相当于“2D 图像的一层”，整体构成“3D 数据立方体”。
3.  **协同滤波**：在这个三维数组上执行稀疏变换（如小波、DCT），然后进行 **[[硬阈值收缩]]或维纳滤波**。由于所有相似块共享相同的结构信息，在变换域中，**真实的信号会集中到少数几个系数上，而噪声则会均匀分散**。
4.  **聚合**：将去噪后的三维块拆开，每个位置的重叠块进行加权平均，得到最终干净的图像。

---

### 🎯 第五层：为什么在 GelSight 和 TacForeSight 中特别有价值？

在 GelSight 的触觉图像中，非局部自相似性表现得非常明显：

| 自相似类型 | 在 GelSight 图像中的体现 | 对去噪/预测的意义 |
| :--- | :--- | :--- |
| **1. 形变模式重复** | 同一压力下，凹陷形状可能在图像不同区域重复出现（如对称结构的按压）。 | 可以利用远程的相似凹陷来校正当前区域的噪声，提高三维重建精度。 |
| **2. 纹理重复** | 接触面的纹理模式是固定的，只是因形变而变形。 | BM3D 能够通过匹配相似纹理块，更好地保留纹理细节。 |
| **3. 时序自相似性** | 相邻帧之间形变是平滑变化的（这才是“时序上的非局部相似性”）。 | GRU 本质上就是在时间维度利用这种“非局部”的相似性来预测未来。 |

---

### 🆚 第六层：在 TacForeSight 中的角色与权衡

虽然 BM3D 作为传统算法在推理阶段（实时性要求高）**无法被直接使用**（计算量太大），但它的思想深刻影响了 TacForeSight 的设计。

| 层面 | 非局部自相似性的体现 | 与 BM3D 的关系 |
| :--- | :--- | :--- |
| **数据预处理** | 对训练集中的图像进行 BM3D 去噪。 | BM3D 作为离线工具，用于清洗训练数据。 |
| **编码器设计** | 卷积神经网络通过逐步扩大感受野来**隐式地捕捉非局部信息**。 | 网络从数据中**学习非局部特征**，不再需要 BM3D 显式计算。 |
| **GRU 时序建模** | GRU 利用**时间上的自相似性**（形变在时间轴上平滑变化）进行预测。 | 这是“时序上的非局部自相似性”利用。 |
| **推理阶段** | 不使用 BM3D（速度慢），但编码器已学会**隐式去噪**。 | 计算效率被放在首位。 |

---

### 💎 一句话总结

**非局部自相似性**是指图像中**不同位置的图像块可能拥有相似的纹理或结构**，这种冗余信息为高级去噪算法（如 BM3D）提供了核心依据。在 GelSight 触觉图像中，形变模式和纹理结构的重复性使 BM3D 能够实现卓越的去噪效果，从而提升三维形貌重建的精度。在 TacForeSight 系统中，虽然 BM3D 因计算成本过高无法用于实时推理，但它被有效地应用在**离线数据预处理**阶段，用以生成高质量的训练数据集；同时，**端到端的编码器通过卷积层和 GRU 的时序建模，在推理阶段以“数据驱动”的方式隐式地利用了非局部自相似性**，实现了计算效率与去噪效果之间的平衡。
